Merknader og definisjoner

★ Sammenslåingsperioder: Følgende fylker var sammenslått i perioden 2018–2023: Vestfold og Telemark (Vestfold + Telemark, 2020–2023, splittet 2024), Viken (Østfold + Akershus + Buskerud, 2020–2023, splittet 2024), Troms og Finnmark (Troms + Finnmark, 2020–2023, splittet 2024) og Trøndelag (Sør- + Nord-Trøndelag, 2018–, ikke splittet). I tillegg ble Agder (2020–), Innlandet (2020–) og Vestland (2020–) opprettet ved sammenslåing. Data for sammenslåingsperioder er markert med avdempet farge i diagrammene og gjelder hele den sammenslåtte enheten.

KOSTRA (KOmmune-STat-RApportering) er et nasjonalt informasjonssystem som gir styringsinformasjon om kommunal og fylkeskommunal virksomhet.

Kilder: Statistisk sentralbyrå (SSB) tabell 07459 (befolkning), 12163 (KOSTRA netto driftsutgifter) og 13561 (finansielle grunnlagsdata). Økonomidata 2020–2025 er verifisert fra SSB. Tall for 2015–2019 er estimater.

Om Fylkesbarometeret

Denne siden er laget med Claude Code av Åsmund Bakkevig. Det er et hobbyprojekt som ble til for å undersøke hvordan man enkelt kunne aggregere og presentere data ved bruk av kunstig intelligens.

Dersom du har spørsmål, tilbakemeldinger, ros, ris eller annet, send meg gjerne en melding på [laster…].

  • Data fra SSB via JSON-stat2 API
